## Troubleshooting

### "No wallet detected"
**Solution:** Install Lace, Eternl, Nami, or Yoroi wallet extension/app first

### "Connection failed"
**Solution:** Refresh page, ensure wallet is unlocked, try again

### Mobile not working
**Solution:** Use Lace Mobile or Eternl Mobile app - they have best mobile support
